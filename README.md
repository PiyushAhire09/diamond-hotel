Overview
This project is a responsive hotel management website built using ReactJS for the frontend and Spring Boot for the backend. It is designed to enhance the user experience for both customers and administrators by providing a comprehensive and intuitive platform for hotel management.

Features
Responsive Design: The responsive website, ensuring it works seamlessly across different devices and screen sizes.
User and Admin Roles: Provides separate interfaces and functionalities for users/customers and admins.
Secure Authentication: Implements robust authentication mechanisms to ensure that only authorized users can access specific parts of the application.
Booking Management: Users can browse available rooms, book rooms, and manage their bookings.
Admin Dashboard: Admins can manage room availability, view customer details, and oversee bookings.


Technology Stack
Frontend: ReactJS
Backend: Java Spring Boot
Database: MySQL & AWS
Build Tool: Maven
